# Mobile Web

Folder ini disiapkan untuk aplikasi penumpang yang mobile-first.

Tujuan utama:

- cari jadwal
- lakukan booking
- bayar via Midtrans
- lihat tiket dan riwayat transaksi

Catatan:

- belum discaffold pada step ini
- dibangun setelah fondasi backend/backoffice cukup stabil
- UI penumpang harus tetap terpisah dari Filament panel
